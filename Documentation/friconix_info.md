## Basics

Use the HTML `<i>` tag with the `fi` prefix, icon parameters and icon name:

`<i class="fi-parameters-iconname"></i>`

### Parameters

`fi-snsuxh-iconname`

__Shape__
* t - triangle
* e - equilateral
* c - circle
* s - square
* h - exagon
* o - octagon
* x none

__Thickness__
* t - thin
* n - normal
* w - wide
* x - none

__Style__
* l - line
* s - solid
* p - prohibited
* x - none

__Direction__
* u - up
* r - right
* d - down
* l - left
* x - none

__Effect__
* h - horizontal
* v - vertical
* s - spin
* p - pulse
* x - none

__Size__
* t - tiny
* s - small
* m - medium
* l - large
* x - extra-large
* h - huge
* 2 - 2x
* ...
* 9 - 9x
